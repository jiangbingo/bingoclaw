# Bingoclaw 技能文档

## 翻译技能 (translate)

### 功能
- 多语言翻译（中英日韩 + 法德西）
- 语言自动检测
- 翻译缓存（localStorage，1小时过期）
- 完善的错误处理

### 使用方法

```bash
# 中译英（自动检测）
翻译 你好世界

# 英译中（自动检测）
translate Hello World

# 指定目标语言
翻译 to ja 你好    # 日语
翻译 to ko 你好    # 韩语
翻译 to fr Hello  # 法语
```

### 返回格式
```
🌐 翻译结果

中文 → 英语

Hello World
```

### 支持语言
- `en`: 英语
- `zh`: 中文
- `ja`: 日语
- `ko`: 韩语
- `fr`: 法语
- `de`: 德语
- `es`: 西班牙语

### 错误处理
- ✅ 空文本检测
- ✅ 文本长度限制（5000 字符）
- ✅ 不支持的语言提示
- ✅ API 错误处理
- ✅ 网络异常处理

## 天气技能 (weather)

### 功能
- 实时天气查询
- 3 天天气预报
- 城市模糊匹配
- 完善的错误处理

### 使用方法

```bash
# 查询天气
天气 北京
weather Shanghai

# 模糊匹配
天气 北    # 自动匹配到北京
天气 上    # 自动匹配到上海
```

## 技术实现

### 翻译技能
- **API**: BigModel API (GLM-4)
- **缓存**: localStorage (1小时过期)
- **语言检测**: 正则表达式 + 字符范围判断
- **文件**: `packages/skills/src/translate/index.ts`

### 天气技能
- **API**: wttr.in (免费，无需 API Key)
- **数据格式**: JSON
- **城市匹配**: 字典映射 + 模糊匹配
- **文件**: `packages/skills/src/weather/index.ts`
